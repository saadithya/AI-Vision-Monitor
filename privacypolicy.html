Privacy Policy

Effective Date: August 2025

This Privacy Policy explains how AI Vision Monitor handles your data when using the Chrome Extension.

Information We Collect

  • Webcam Access: Used only when you open the camera page to perform AI-based object detection. Images are processed locally or sent securely to your chosen AI provider (Google Vision or OpenAI).
  • API Keys: Stored locally in your browser using chrome.storage.local. We do not transmit or store them outside your device, except to make requests to the chosen provider.
  • License Keys: Verified with Gumroad’s API for activation. Keys are stored locally.
  • Usage Data: The extension does not collect analytics, browsing history, or personal data.

How We Use Your Information

  • Provide AI object detection features
  • Validate trial and license activation
  • Enable smart notifications

Third-Party Services

The extension communicates with:

  • Google Vision API – if selected as AI provider
  • OpenAI API – if selected as AI provider
  • Gumroad API – to validate licenses

These services have their own privacy policies.

Data Retention

All data (trial info, API keys, license keys, preferences) is stored locally in your browser. You can clear it anytime by uninstalling the extension or clearing Chrome storage.

Your Rights

You may uninstall the extension at any time. No data is retained by us after uninstallation.

Contact

If you have any questions, please contact us through our GitHub repository.
