AI Vision Monitor

AI Vision Monitor is a Chrome Extension that brings AI-powered webcam monitoring to your browser. It uses Google Vision or OpenAI to detect objects and provide smart alerts in real time.

Features

  • AI-powered object detection from your webcam feed
  • Choose between Google Vision or OpenAI as your AI provider
  • Free trial with optional license activation
  • Smart notifications when objects are detected
  • Easy setup with API key validation

Installation

You can install AI Vision Monitor directly from the Chrome Web Store.
